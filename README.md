A single-page portfolio website for BrandBroz Media.

```
brandbroz-portfolio/
├── index.html          ← the whole portfolio (HTML + CSS + JS in one file)

│   ├── svs-foods.mp4
```

## 🚀 Deploy in 2 minutes (FREE, public link)

### Option 1 — Netlify Drop (easiest, no signup required for testing)

1. Go to **https://app.netlify.com/drop**
2. Drag the entire `brandbroz-portfolio` folder onto the page
3. Done. You'll get a public link like `https://random-name-12345.netlify.app`
4. (Optional) Sign up free → claim the site → rename to `brandbroz.netlify.app`

### Option 2 — Vercel (also great)

1. Go to **https://vercel.com/new**
2. Click "Deploy" → drag-drop the folder
3. Free permanent URL.

### Option 3 — GitHub Pages (free forever, with your own domain)

1. Create a GitHub account
2. New repo → name it `brandbroz-portfolio`
3. Upload all the files from this folder
4. Settings → Pages → Source: `main` branch → Save
5. Your site goes live at `https://YOUR-USERNAME.github.io/brandbroz-portfolio`

### Option 4 — Cloudflare Pages (fastest worldwide)

1. Go to **https://pages.cloudflare.com**
2. Upload the folder → get a `.pages.dev` URL.

**My pick: Netlify Drop.** Fastest, no friction, zero configuration.

## ✏️ How to edit

Open `index.html` in any text editor (VS Code, Sublime, even Notepad). Everything you might want to change is in plain text:

| Want to change... | Find this in index.html |
|---|---|
| Hero headline | `<h1>We make brands...` |
| Stats (7.3M, 944K, etc) | Inside `<div class="stats">` |
| Team members | `<section id="team">` |
| Services | `<section id="services">` |
| Client list | Inside `class="clients-grid"` |
| Phone / email / address | `<section class="contact-section">` |
| Colors | Top of `<style>` — change `--accent: #d4ff3a` |
